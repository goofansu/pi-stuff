# AGENTS.md

## System management

- This system is managed with Nix. Do not install software automatically using Homebrew, pip, npm, or any other package manager.
- If a required tool is missing, tell the user what is needed and let them install it themselves.

## Commands

- Do not use interactive TTY commands.
- Always use python3 to execute Python scripts.

## Writing

- Use sentence case for headings.
